A Simple and Flexible WebRTC Platform

WebRTCは、ブラウザ同士でリアルタイムに映像、音声、データを送受信できる画期的な新技術です。SkyWayを使えば、サーバを準備することなく、簡単なJavaScriptを書くだけで、WebRTCを使ったアプリが開発できます。

リアルタイム

リアルタイムにデータを送受信できます。ネットゲームやファイル転送ソフトに活用できます。

マルチデバイス&ブラウザ

PCでもスマートフォンでも、WebRTC対応ブラウザさえあれば、共通のコードが動作します。

マルチメディア

映像と音声を送受信でき、カメラやマイクも利用できます。ビデオチャットなどを簡単に作ることができます。

インストールなし

ブラウザでURLにアクセスするだけで使えます。アプリのインストール作業は一切不要です。

今すぐAPIキーを申し込む (無料)

使ってみよう

  1. SkyWayのWebAPIを利用するため、APIキーを申し込みます。
    APIキーを申し込む (無料)
  2. ライブラリをインクルードし、JavaScriptコードを書きましょう。 
    <script src="https://skyway.io/dist/0.3/peer.js"></script>
<script>
var peer = new Peer(myId, {key: 'yourApiKey'});
var conn = peer.connect('anotherId');
conn.on('open', function() {
    conn.send(prompt('Input message'));
});
</script>
</body>
