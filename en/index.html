A Simple and Flexible WebRTC Platform

WebRTC is a revolutionary new technology that allows browsers to send video, audio and data to other browsers in real time. If you use SkyWay, you can easily create WebRTC apps without having to worry about the server.

Real time

Send data in real time. It can be used in online games or file transfer software.

Multidevice & browser

Use the same code to run it on any WebRTC compatible browser*, regardless of whether it's on a PC or smart phone. It can also be incorporated into smartphone native app.
*Currently supported on Chrome(excluding iOS)and Firefox

Multimedia

You can send and receive video and audio from any compatible camera or microphone connected. You can make things like video chat apps easily.

No installation needed

Just access the URL in your browser to start. There's no need for any complicated installation.

Using SkyWay

SkyWay provides a PeerServer compatible Web API, a modified PeerJS JavaScript library, iOS SDK and Android SDK.

  1. Sign Up for SkyWay and apply for an API key to use the SkyWay WebAPI
    Sign Up for SkyWay (free)
  2. Developing an application using the SkyWay.

    In case of using JavaScript library

    Refer to the code sample, include the library and write the JavaScript code.
    <script src="https://skyway.io/dist/0.3/peer.min.js"></script>
<script>
var peer = new Peer(myId, {key: 'yourApiKey'});
peer.on('open', function(id) {
    var conn = peer.connect('anotherId');
    conn.on('open', function() {
        conn.send('hello world');
    });
)};
</script>

    In case of using iOS SDK

    After downloading the SDK, add SkyWay.framework to your Xcode project and import header file.
    Download the iOS SDK（0.1.0）Sample App for iOS

    In case of using Android SDK

    After downloading the SDK, Add SkyWay.aar to your application and import class files.
    Download the Android SDK（0.1.0）Sample App for Android

    ※ In case of WebRTC communication with JavaScript library and iOS/Android SDK, please check here notes.
